浮点乘法取反指令

fnmpy

操作（类似 fnma） 字体惯例

指令类型 F

格式

(qp) fnmpy.pc.sf f1 = f3, f4 pseudo-op of: (qp) fnma.pc.sf f1 = f3, f4,f0

说明

f3 与 FR f4 的乘积计算到无限精度，然后取反。随后按照 FPSR.sf.rc 指定的取整模式，将结果值取整到 pc（及可能的 FPSR.sf.pc 与 FPSR.sf.wre）指定的精度。取整后的结果放入 FR f1。

如果 FR f3 或 FR f4 是 NaTVal，则将 NaTVal 放入 FR f1，而不放入计算的结果。

如需了解状态字段的 pc、wre 以及 rc 的编码与解释，请参阅浮点计算控制定义与浮点取整控制定义。
